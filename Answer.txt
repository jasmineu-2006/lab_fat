# 🟢 STEP 4: DOCKERIZE APP

👉 Create `Dockerfile`

```dockerfile
FROM nginx:alpine

COPY . /usr/share/nginx/html

EXPOSE 80
```

---

## 🔹 BUILD & RUN

```bash
docker build -t university-app .
docker run -p 8080:80 university-app
```

👉 Open:

```

```

# 🟢 STEP 6: JENKINS PIPELINE

👉 Create Pipeline → paste:

```groovy
pipeline {
    agent any

    stages {
        stage('Clone') {
            steps {
                git branch: 'main', url: '<repo-link>'
            }
        }

        stage('Build Docker') {
            steps {
                bat 'docker build -t university-app .'
            }
        }

        stage('Push DockerHub') {
            steps {
                bat 'docker push <username>/university-app:v1'
            }
        }
    }
}
```

👉 Click **Build Now**

---

# 🟢 STEP 7: KUBERNETES DEPLOYMENT

👉 `deployment.yaml`

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: university-app
spec:
  replicas: 1
  selector:
    matchLabels:
      app: university-app
  template:
    metadata:
      labels:
        app: university-app
    spec:
      containers:
      - name: university-app
        image: <username>/university-app:v1
        ports:
        - containerPort: 80
```

---

# 🟢 STEP 8: SERVICE

👉 `service.yaml`

```yaml
apiVersion: v1
kind: Service
metadata:
  name: university-service
spec:
  type: NodePort
  selector:
    app: university-app
  ports:
    - port: 80
      targetPort: 80
      nodePort: 30020
```

---

# 🟢 STEP 9: DEPLOY

```bash
kubectl apply -f deployment.yaml
kubectl apply -f service.yaml
```

---

# 🟢 STEP 10: RUN

```bash
kubectl port-forward service/university-service 8080:80
```

👉 Open:

```
http://localhost:8080
```
